Usage: harness [OPTIONS] COMMAND ...

Options:
  -C, --color    Enable ANSI color output
  -N, --no-color Disable ANSI color output
  -h, --help     This tool summary

Commands:
           collect: Collect key and value
         configure: Configures the system for some thing
  default-variants: Different option defaults
              help: List available commands
          in-order: Execute remote command
          set-mode: Sets the execution mode
         tool-info: Echoes the tool name and root command map keys
          validate: validate command
